Contents of the folder:

- rawdata_revision_tidy.R -> script for data cleaning
- subjsim_rater_grouping_tidy.R -> script creating the stimulus groups for the subjective rating groups
- recall_visualizer_subjsim.R -> visualizing script tailored to the cleaned data from the revision script
- image_pool -> folder with the pictured generated by the visualizer
- sample -> folder with pictures from other condition used as examples 6 pcs
- prototypes -> folder with the prototypes images 4 pcs
- 1-4.csv files -> files of the grouped stimuli created by the grouping script, used by the OS and jzip files
- subjsimrating1-4_group1-4.OS files -> Opensesame experiment files for the groups
- subjsimrating_group1-41-4.jzip files -> OS files converted for the JATOS platform
