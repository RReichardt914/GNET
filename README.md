stimulus_generator - scripts that generate the stimulus pool for the task

vispatterns - this is the original task, later named "GNET". contains several rounds of data collection. lab recorded task and a version with 3d stimuli

vispatterns_v2 - modified task - difference between images is varied systematically leading to stepwise increase - interesting pattern in recognition (middle values remembered with a higher prob)

gnet-4stim - reduced number of images, delay btwn study and test, narrative instruction

gnet_recall_analysis - recall test data from "gnet-4stim"
